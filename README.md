# Foco · Pomodoro 🍅

App de temporizador Pomodoro gamificada, minimalista y sin distracciones, para móvil y tablet.

**Ábrela aquí:** https://javidym.github.io/foco-pomodoro/

Una vez abierta en el navegador del móvil o tablet, usa **"Añadir a pantalla de inicio"** para instalarla como app. Funciona sin conexión y todo tu progreso se guarda en el propio dispositivo.

## Características
- Tiempos 20/5 con descanso largo cada 4 pomodoros (configurable).
- Pantalla sin distracciones: los controles se ocultan mientras trabajas.
- Frases motivadoras pasivas y sutiles.
- **Sobreesfuerzo**: si sigues tras acabar el foco, ganas fichas y acumulas descanso.
- Fichas, niveles y rangos.
- Tienda de bonus (recorte de tiempo, multiplicadores, descanso doble, escudo de racha…).
- Rachas diarias y logros.
- Estadísticas: hoy, semana, mes, total, con gráficos.

Un único archivo `index.html`, sin dependencias.
